Disk A3087A (2 GB, C2490AM)

3.5-inch SE Hard Disk Drive Coniguration

Caution

  • The drive is set for the system in which it is installed. Changing the configuration may cause the drive to malfunction.
  • Devices outside the system will be configured differently.
  • The connector jumpers are not interchangeable from one connector to another. Interchanging the jumpers may damage the connectors.

The driver configuration is set with jumpers on the address/option connector and option connector (see Figure) shows how to the option pin-sets for the A3087A 3.5-inch hard disk drive should be set. A pin-set is shorted with a jumper installed, or open without a jumper.

  • Disk Size/Capacity : 3.5-inch 2 GB
  • Productnumber : A3087A
  • Vendor/Model Number : C2490AM
  • Exchangenumber : C2490-69365
  • Interface Type : Single-Ended
