Disk A3058A (1 GB, ST31200WD)

3.5-inch WD Hard Disk Drive Configuration



Caution
  • The drive configuration is set for the system in which it is installed. Changing the configuration may cause the drive to malfunction.
  • Devices outside the system maybe be configured differently.
  • On Dec/Quantum products, the jumpers are interchangeable between connectors. On the Seagate model connector J2 jumpers are unique.


A3058A represents two interchangeable 1GB low profile disk drives:

  • Seagate model number ST31200WD
  • Dec/Quantum model number DSP3107LWD

The drives are fixed, sealed hard drives intended for primary mass storage applications. These are SCSI-2 compliant fast-wide devices with embedded controllers. The drives are functional and replacement equals. To see the information provided below references manufactuaring product number, click here DSP3107LWD.html .


A3058A Hard Disk Drive (Manufactuaring Product# ST31200WD)

The drive options are selected on J2. The drive SCSI ID (address) is set on either J5 or J6 (see the following Figure 1). The following table 1 shows the pin-sets for the options available with the A3058A 3.5-inch hard disk drive (MFG Product # ST31200WD). The actual settings depend upon the system in which the disk drive is installed and its location within that system. See your service manual or CE handbook for specific configuration and jumper setting information.

Table 2 provides a functional description of the J2 jumpers.

A pin-set is either shorted (with jumper installed) or open (without a jumper). The J5 and J6 option settings are on Figure 1.
