Disk A2958A (535 MB, ST3610N)

3.5-inch SE Hard Disk Drive Configuration

Caution

  • The drive configuration is set for the system it is installed in. Changing the configuration may cause the installed in. Changing the configuration may cause the drive to malfunction.
  • Devices outside the system will be configured differently.
  • The connector jumpers are not interchangeable from one connector to another. Interchanging the jumpers may damage connector to another. Interchanging the jumpers may damage the connectors.

  • Disk Size/Capacity : 3.5-inch 533MB
  • Vendor/Model Number : ST3610N
  • Exchangenumber : A2958-69001
  • Interface Type : Single-Ended

  • 8x7, G, H, I Class systems.

The drive options are selected on J2. The drive SCSI ID (address) is set on J5 (see Figure). Table 1 shows how the option pin-sets for the 3.5-inch hard disk drive (A2958A) should be set. Table 2 provides a functional description of the J2 jumpers.

A pin-set is shorted with a jumper installed or open without a jumper. The J5 option settings are shown on the following Figure.
