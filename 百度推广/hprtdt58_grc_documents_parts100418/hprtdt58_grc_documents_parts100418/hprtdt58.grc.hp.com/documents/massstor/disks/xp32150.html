Overview

The Quantum Atlas™ Series 3.5-inch, 7,200 RPM hard disk drives meet the demanding performance and reliability requirements of high-performance, high-capacity applications. They are targeted at mission-critical, data-intensive system environments where 24-hour-a-day availability is essential.

Systems that can benefit from the Atlas family's exceptional performance include RAIDs (Redundant Arrays of Independent Disks), network file servers, supercomputers, mainframes, database servers, workstations, high-performance PCs, desktop publishing and video systems, and audio/video servers.

Industry-leading acoustic performance and low operating power make the Atlas drives a superb choice for high-end desktop applications. Optimized performance features include a custom SCSI controller, a sophisticated command reordering algorithm, an embedded quadrature servo system, and a large multi-track segmented cache.

Advanced SCSI Architecture and Cache Performance

Quantum's custom ASABET™ SCSI controller provides exceptionally low command overhead and reduced SCSI bus execution times, as an unprecedented number of functions are automatically executed in hardware. The company's Three-Dimensional Rotationally Optimized Seek Execution (3-D ROSE™) algorithm also improves performance by reordering commands to minimize seek and rotational delay.

The Atlas Series drives feature a large cache and a unique cache-management algorithm that maximizes cache hits for multiple sequential read streams. The result is fast data access and high system throughput.

Quantum's Write Back Cache improves write performance by appending multiple consecutive write commands into a single-revolution write operation. SCSI bus resets do not destroy cached write data that has not been written to the media.

Multimedia Supported

The Atlas drives are well-suited for real-time multimedia applications.

These Quantum drives employ an advanced embedded quadrature servo system that continuously compensates for temperature changes, eliminating the need for thermal recalibrations that interrupt audio/video data streams.

Exceptional Data Integrity and Reliability

Quantum drives have the highest data integrity in the industry. Data from the SCSI bus to the media is protected with an Error-Detection Code (EDC). Additionally, Quantum incorporates a powerful Reed-Solomon Error Correction Code (ECC) capable of correcting a single 73-bit error burst for each sector.On-the-fly ECC can correct a single 25-bit error burst for each sector.

The Atlas drives alert the host system with an Early Warning Status (EWS) if drive performance approaches unacceptable reliability levels. This minimizes operational impact by allowing time for preventive service.
