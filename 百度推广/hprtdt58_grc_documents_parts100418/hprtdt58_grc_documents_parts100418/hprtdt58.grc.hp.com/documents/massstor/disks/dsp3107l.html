Disk A3182A (1 GB, DSP3107L)

3.5-inch SE Hard Disk Drive Configuration

Caution

  • The drive is set for the system in which it is installed. Changing the configuration may cause the drive to malfunction.
  • Devices outside the system will be configured differently.
  • On Dec/Quantum products, the jumpers are interchangeable between connectors. On the Seagate models connector J2 jumpers are unique.

A3182A represents two intechangeable 1 GB low profile disk drives:

  • Seagate model number ST31200N
  • Dec/Quantum model number DSP3107L
These drives are fixed, sealed hard drives intended for primary mass storage applications. These drives are SCSI-2 compliant single-ended devices with embedded controllers. The drives are functional and replacement equals. The information provided below references manufacturing product number ST31200N.

A3182A Hard Disk Drive (MFG Product# DSP3107L)

The drive options and SCSI ID (address) are set on J3 (see the following Figure). Table 1 shows the pins- sets for the options available with the A3182A 3.5-inch hard disk drive (MFG Product# DSP3107L). The actual settings depend upon the system in which the disk drive is installed and its location within that system.

Table 2 provides a finctional description of the J3 jumpers. A pin-set is either shorted (with a jumper installed) or open (without a jumper).

  • Disk Size/Capacity : 3.5-inch 1 GB
  • Productnumber : A3182A
  • Vendor/Model Number : DSP3107L
  • Exchangenumber : A3182-69001
  • Interface Type : Single-Ended

    Note

    Write Protect mode and the Spin Delay mode are supported but not simultaneously. The present drive mode is determined by the state of the Write Protect mode page bit. The factory defaults is Spin Delay mode.
