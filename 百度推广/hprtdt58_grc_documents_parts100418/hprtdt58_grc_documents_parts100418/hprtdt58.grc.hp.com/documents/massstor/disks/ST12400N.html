Disk A3191A (2 GB, ST12400N)

3.5-inch SE Hard Disk Drive Configuration

Caution

  • The drive configuration is set for the system in which it is installed. Changing the configuration may cause the drive to malfunction.
  • Devices outside the system will be configured differently.
  • The connector jumpers are not interchangeable from one connector to another. Interchanging the jumpers may damage the connectors.

The drive options are selected on J2. The drive SCSI ID (address) is set on J6 (see floowing figure). Table 1 shows how the option pin-sets for the A3191A 3.5-inch hard disk drive should be set. Table 2provides a functional description of the J2 jumpers.

A pin-set is either shorted (with a jumper installed) or open (without a jumper). The J6 option settings are shown on the following Figure.

  • Disk Size/Capacity : 3.5-inch 2 GB
  • Productnumber : A3191A
  • Vendor/Model Number : ST12400N
  • Exchangenumber : A3191-69001
  • Interface Type : Single-Ended
