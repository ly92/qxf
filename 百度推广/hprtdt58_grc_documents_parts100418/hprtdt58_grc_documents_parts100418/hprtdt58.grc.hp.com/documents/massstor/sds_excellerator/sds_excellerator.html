Solid Data Disk Excellerator

Overview Features
Excellerators are a family of self-contained solid state storage systems that use non-volatile SDRAM technology. The Excellerator Ultra family is designed for applications with a high rate of usage such as digital media, internet services, relational databases, and on-line transaction processing (OLTP). All four models have an industry-standard SCSI interface and support all major Unix and Microsoft NT platforms.
An internal backup disk ("HD Cannister") is used to store the data in case of power failure or shutdown.
  • Provides high speed access to I/O intense Unix and NT data files
  • Enhances the performance of existing RAID and JBOD storage systems
  • Scales from 536MB up to 34Gb, depending on the model
  • System self-configures to number of array boards installed
  • Up to 15 systems can be configured on a single SCSI Wide bus
  • Incorporates a non-volatile system architecture
  • Redundant power supplies (800 and 1000 Ultra only) and a hot-swappable backup disk drive
  • Backup and restore data rate of 536MB/min
